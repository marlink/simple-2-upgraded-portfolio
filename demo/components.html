Tabs

Matches the structure in `components.js`. Arrow keys move between tabs and the active panel receives `.is-active` while other panels remain hidden visually.

Tabs are keyboard navigable and manage ARIA automatically.

  • Use `.tab` buttons inside `.tablist`.

Modal

Dialogs open via `data-modal-target` and close with `data-modal-close`, backdrop clicks, or the Escape key. Focus is trapped inside the dialog until dismissal.

Launch modal

Markup blueprint

<button data-modal-target="#demo-modal-basic">Open modal</button>
<div class="modal" id="demo-modal-basic" hidden>
    <div class="modal__overlay"></div>
    <div class="modal__dialog">
        <button data-modal-close aria-label="Close modal">Close</button>
        <h2>Modal Title</h2>
        <p>Modal content goes here.</p>
    </div>
</div>

Copy this pattern for each modal. IDs must remain unique.
